# 期中作業  
本作業包含三個實作題，範圍涵蓋 HTML 結構、表格與表單元素。

## 實作題一：個人簡介網頁  
建立一個名為 profile.html 的個人簡介網頁，內容包含：
1. `<head>` 內設定 `<meta charset="utf-8">`、`<title>` 與 `<style>`，讓主標題（h1）呈藍色。  
2. `<body>` 包含姓名、兩段自我介紹、水平線、聯絡資訊（含 mailto 超連結），並以 `<div>` 包裹整體內容。  

提示：  
- 使用 `<meta>`、`<style>` 與 `<a>` 超連結。
- 可參考書中 4-1 至 4-5 節。
- `h1 {color: blue;}` 藍色

## 實作題二：商品資訊表格  
建立一個名為 product.html 的網頁，用表格呈現三項商品資訊：
表格需包含 `<caption>`、`<thead>`、`<tbody>`、`<tfoot>` 結構，邊框屬性使用 `frame="border"` 與 `rules="all"`。

提示：  
- 使用 `<th>` 標示表頭。  
- 可參考書中 4-8 節範例。  

## 實作題三：意見回饋表單  
建立一個名為 feedback.html 的網頁，製作一份簡單的表單，內容包含姓名、電子郵件、問題描述與分類下拉選單，並附提交與重設按鈕。
提示：
- 使用 `<form>`、`<label>`、`<input>`、`<textarea>`、`<select>`、`<button>`。
- 參考書中 4-9 節範例。
